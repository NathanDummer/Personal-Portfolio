NextCloud

The goal for this project was to create an application for the IT which could automatically create users, delete users, create groups, and create shares for the groups. As a result we save roughly $50,000 annually as we were able to use an alternative file sharing service. I used Selenium in Python for all of the automation aspects and logic. Flask was used for the dynamic front-end. This project had many pieces and required logging, automatic email generation, secure password generation, and unit testing. Overall, it was a challenging project and I learned a lot.


JBOD Hardware Simulator

Developed a software-based JBOD simulator in C to emulate storage device operations including reading, writing, and data retrieval functionality. Implemented makeshift cache functionality and also configured a socket-based communication system. Overall I applied low-level systems programming techniques to handle disk management, data organization, and error handling.


Course Scheduler

For my OOP class I created a course scheduling application. Using Java, I created a GUI and connected it to a Derby Database. Using Prepared Statements and OOP techniques I created functions for sorting, deleting, and adding to the database.
