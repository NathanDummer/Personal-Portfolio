NextCloud

The goal was an app that the IT team could use to share files and delete users in a secure and simple manner. We also used it to migrate from another file sharing service to save around $4,000 a month. I used Selenium in Python to run the automation. I also used React within Flask for the front-end portion. The biggest challenge for this project was coming up with creative solutions for dealing with unpredictable web elements. There was also a huge focus on error handling and making sure the processes were completed correctly.


Course Scheduler
